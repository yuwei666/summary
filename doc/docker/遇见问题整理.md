问题描述：

​	容器内无法访问宿主机端口8082，但非容器内通过ip:8082是可以正常访问的。

问题原因：

​	检查宿主机是否允许从 `docker0` 网卡访问：

```bash
sudo iptables -L -n | grep docker0
```

​	如果无相关 `ACCEPT` 规则，手动添加：

```bash
# 允许从 docker0 访问宿主机端口
sudo iptables -A INPUT -i docker0 -j ACCEPT

# 精确放行，允许从 docker0 访问宿主机的 8082 端口（gateway）
sudo iptables -A INPUT -i docker0 -p tcp --dport 3306 -j ACCEPT
```

补充，正常情况下，容器间相互访问，用 容器名称:port 访问，也可以实现。
